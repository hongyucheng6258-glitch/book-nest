# JavaEE后端的期末作业

## 目的
1. 检验对SpringBoot框架的掌握程度
2. 检验根据需求对SpringBoot/SpringSecurity进行二次开发的能力
3. 检验对根据需求对系统进行二次开发的能力
4. 检验与前端进行协调开发的能力

## 要求
请为书城App设计对应的后端：
![主页](images/01.png)

### 总体功能

1. 根据类别筛选书籍列表
2. 对书名和作者进行搜索的功能
3. 展示书籍详情
4. 加入购物车的功能
5. 登录及个人信息功能

### 登录交互
1. 需要提供登录页面，可以执行登录，并保存登录状态。
2. 个人信息、购物车等功能需要处于登录状态

### 购物车或关注功能
1. 购物车/关注数据应存放于后端，使用ajax交互，数据可以增减。
2. 购物车应该可以计算总值。
3. 可以移除单个商品。

### 搜索
1. 你需要设计一个搜索页面，给出用户的查询结果。
2. 页面上的商品可点击，点击进入详情页面。

### 个人信息
1. 你要有一个个人信息页面
2. 个人信息应该可以查看购物订单或收藏的信息

(以上的部分可以获得80分，以下为提高分)

### 购物订单
1. 可以对购物车的商品购并生成订单
2. 可以对订单进行查看和管理

### 个人信息MKII
1. 增加用户信息浏览
2. 增加头像设置等功能
